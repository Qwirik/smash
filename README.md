# SMASHOME - Atmospheric Intelligence

Современный проект умного дома. Состоит из высокопроизводительного C++ бэкенда (SmashCore) и красивого современного фронтенда на React/Vite.

---

## 🛠 Предварительные требования

Перед запуском убедитесь, что у вас установлены:

* **Для бэкенда:** `cmake`, `g++`, `libpqxx-dev`, `libspdlog-dev`, `nlohmann-json3-dev`, PostgreSQL сервер, `libasio-dev`, `libreadline-dev`.
* **Для фронтенда:** `Node.js` (рекомендуется v18+) и менеджер пакетов `npm`.

---

## 🚀 1. Запуск Бэкенда (SmashCore)

Бэкенд написан на C++ и использует PostgreSQL.

1. **База данных:**
   Убедитесь, что сервер PostgreSQL запущен, и у вас создана база данных.
   По умолчанию SmashCore пытается подключиться по URL: `postgresql://qwirik:smash_password@localhost/smashcore`.

2. **Сборка проекта:**
   Откройте терминал в корневой папке проекта и выполните:
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```

3. **Запуск:**
   После успешной компиляции запустите сервер:
   ```bash
   ./SmashCore
   ```
   *По умолчанию сервер API запустится на порту `8080`.*

---

## 🎨 2. Запуск Фронтенда (SMASHOME UI)

Фронтенд написан на React с использованием сборщика Vite.

1. **Настройка переменных окружения:**
   Перейдите в папку `frontend`. Убедитесь, что там есть файл `.env` со следующим содержимым:
   ```env
   VITE_API_BASE_URL=http://localhost:8080
   VITE_API_KEY=admin123
   ```
   *(Эти ключи нужны для того, чтобы фронтенд мог отправлять авторизованные X-API-Key запросы к вашему C++ серверу).*

2. **Установка зависимостей:**
   Выполните установку необходимых NPM пакетов:
   ```bash
   cd frontend
   npm install
   ```

3. **Запуск сервера разработки:**
   Для запуска сайта в режиме разработки выполните:
   ```bash
   npm run dev
   ```
   Сайт будет доступен по адресу: [http://localhost:5173](http://localhost:5173) (или по порту, который выдаст Vite).

## 📝 Заметки по API

Если вам потребуется протестировать бэкенд без фронтенда, вы можете использовать `curl`.
Например, для переключения реле:

```bash
curl -X POST http://localhost:8080/api/web/command \
     -H "Content-Type: application/json" \
     -H "X-API-Key: admin123" \
     -d '{"device": "Living_Room_Light", "command": "relay_on"}'
```
