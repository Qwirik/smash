# Инструкция по запуску проекта (Windows)

Этот проект состоит из двух частей: бэкенда (Node.js) и фронтенда (React + Vite).

## Предварительные требования
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия LTS).

---

## 1. Запуск бэкенда (Backend)
1. Откройте терминал в папке `backend`.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   node index.js
   ```
   *Сервер будет запущен на [http://localhost:3000](http://localhost:3000).*

---

## 2. Запуск фронтенда (Frontend)
1. Откройте **второе окно терминала** в папке `frontend`.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите клиентскую часть:
   ```bash
   npm run dev
   ```
   *Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173).*

---

## Проверка работы
1. Перейдите по адресу [http://localhost:5173](http://localhost:5173) в браузере.
2. Введите любой email на странице входа.
3. Вы должны увидеть панель управления (Dashboard) со списком устройств.
