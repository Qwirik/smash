# Инструкция по запуску проекта на локальном компьютере

Чтобы запустить "Smart Home Dashboard" у себя на компьютере, выполните следующие шаги:

### 1. Подготовка
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18 или выше).

### 2. Запуск Бэкенда (Серверной части)
Сервер отвечает за API и хранение данных.
1. Откройте терминал (командную строку).
2. Перейдите в папку `backend`:
   ```bash
   cd backend
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   node index.js
   ```
   Вы должны увидеть сообщение: `Server started on http://localhost:8080`.

### 3. Запуск Фронтенда (Интерфейса)
Фронтенд — это визуальная часть сайта.
1. Откройте **второе** окно терминала.
2. Перейдите в папку `frontend`:
   ```bash
   cd frontend
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
5. В терминале появится ссылка, обычно это `http://localhost:5173`. Нажмите на неё или скопируйте в браузер.

### 4. Вход в систему
1. На странице авторизации введите API Ключ: `admin123`.
2. Нажмите "Войти".

Теперь вы можете просматривать список устройств и управлять ими!
